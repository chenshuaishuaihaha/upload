# Upload

## Build the main environment

```shell
docker build -t upload-labs .
```

## Build a single environment

```shell
Enter a single directory and 
docker build -t upload/pass:1(your subject) .
```

# 文件上传靶场

## 建立主环境

```shell
docker build -t upload-labs .
```

## 建立单个环境

```shell
进入文件目录
docker build -t upload/pass:1(your subject) .
```
